## 基础作业（必做）

### 任务描述

编写一个交互式的个人名片生成器程序 `business_card.py`。

### 功能要求

1. 使用 `input()` 获取以下信息：
   - 姓名
   - 年龄
   - 职业

2. 使用 f-string 格式化输出一张美观的名片，包含：
   - 装饰边框（可使用 `=`、`-`、`*` 等字符）
   - 居中的"个人名片"标题
   - 对齐的信息展示
   - 年龄可以自动添加"岁"字作为后缀

3. 输出示例：

```
==============================
        个人名片
==============================
姓名：李小明
年龄：25 岁
职业：Python 学习者
==============================
```

### 输入/输出示例

**输入：**
```
请输入姓名：李小明
请输入年龄：25
请输入职业：Python 学习者
```

**输出：**
```
==============================
        个人名片
==============================
姓名：李小明
年龄：25 岁
职业：Python 学习者
==============================
```

### 实现提示

**步骤 1：获取用户输入**
- 使用 `input()` 获取姓名、年龄、职业

**步骤 2：格式化输出**
- f-string 对齐语法：`{变量:<宽度}` 左对齐，`{变量:^宽度}` 居中
- 边框技巧：`"=" * 30` 生成 30 个等号

**步骤 3：处理年龄**
- `input()` 返回字符串，可以直接拼接 "岁" 字，如 `f"年龄：{age} 岁"`

### 常见错误

小北第一次写完后运行，发现年龄显示有点奇怪：

```
年龄：25 岁 岁
```

检查代码发现他在 f-string 里写了 `{age} 岁`，但 age 变量本身已经包含 "25" 这个字符串，不需要额外处理。或者如果 age 是纯数字字符串，注意别重复添加单位。

### 任务描述

```

## 挑战作业（选做）

### 任务 1：ASCII 艺术边框

使用特殊字符（如 `╔╗╚╝║═` 等制表符）绘制更精美的边框：

```
╔══════════════════════════════╗
║          个人名片              ║
╠══════════════════════════════╣
║  姓名：李小明                  ║
║  年龄：25 岁                 ║
║  职业：Python 学习者         ║
╚══════════════════════════════╝

- `open()` 函数
- `write()` 方法
- `with` 语句（上下文管理器）
- `datetime` 模块获取当前时间
